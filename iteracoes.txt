Brainstorm sobre o projeto. Decidimos desenvolver um site de review de jogos para as mais diversas plataformas.

04/09/2017
Planejamento sobre pré-requistos e funcionalidade do projeto. Decisões tomadas:
 - Quem são os stakeholders (administradores, desenvolvedores e usuários, cadastrados e não-cadastrados)
 - Cada jogo terá uma pequena descrição sobre o jogo, possibilidade de link para a página do jogo.
 - Moderador vai adicionar todos os jogos, mas usuários podem sugerir novos jogos.

09/10/2017
Continuação do desenvolvimento dos cenários.

Desenvolvimento dos protótipos Lo-FI UI.
 - página inicial : aparecerá os últimos reviews por ordem temporal
 - página de avaliação de jogo : terá foto, pré-requisito e avaliação dos usuários sobre o jogo
 - login : onde usuário já cadastrado poderá entrar no sistema
 - cadastro : onde usuário não cadastrado poderão criar uma conta de usuário para entrar no sistema e criar reviews
 - "meus reviews" do usuário : aonde o usuário pode editar ou excluir review
 - editar perfil : aonde usuário poderá editar seus dados
